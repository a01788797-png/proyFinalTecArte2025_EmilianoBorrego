Nuestro proyecto consta de dos programas. El primero llamado funciones.py , que toma una serie de datos de nuestro otro documento y
devuelve el área de una figura, este programa solo define las funciones que harán los cálculos necesarios para nuestro otro programa.
Después nuestro segundo programa, figurasPrincipal.py, lee de un documento csv una serie de datos, el primero es una figura 
(triángulo, círculo y rectánculo), y el segundo y tercer dato son parámetros de esa figura (base, altura o radio).
después de leer el documento toma esos datos y los despliega a manera de lista cada uno, seguido de su área respectivamente.

Para ejecutar escribe:
python3 -m venv /home/borrego/proyectoFinalArte
